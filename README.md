# Keer op Keer 2

An online multiplayer browser clone of the roll-and-write dice game Keer op Keer 2.

**Play at: https://keer2.vercel.app/**

## Hosting

- **Frontend**: Deployed on [Vercel](https://vercel.com)
- **Backend**: Database, authentication, and real-time features hosted on [Supabase](https://supabase.com)

## Development

First, run the development server from the `app/` directory:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Commands

All from the `app/` directory:

```bash
npm run dev         # Start dev server on localhost:3000
npm test            # Run all unit tests (vitest)
npm run test:watch  # Watch mode
npm run build       # Production build
```
